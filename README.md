# MVP

A full-stack web application with a Rust backend and SvelteKit frontend.

## Architecture

| Layer    | Technology                           |
| -------- | ------------------------------------ |
| Backend  | Rust, [Axum](https://docs.rs/axum/latest/axum/), OpenAPI v3 |
| Frontend | [SvelteKit](https://svelte.dev/docs/kit) (Svelte 5), TypeScript |

- **`backend/`** — REST API server with auto-generated OpenAPI documentation.
- **`frontend/`** — Web client with file-based routing and server-side rendering.

## Prerequisites

- [Rust](https://www.rust-lang.org/tools/install) (latest stable)
- [Node.js](https://nodejs.org/) (22 or later)

## Getting Started

### 1. Clone the repository

```sh
git clone <repo-url>
cd mvp
```

### 2. Start the backend

```sh
cd backend
cargo run
```

The API server starts at `http://localhost:3000`.

### 3. Start the frontend

```sh
cd frontend
npm install
npm run dev
```

The web app starts at `http://localhost:5173` and proxies API requests to the backend.

## Useful Commands

| Command | Description |
| ------- | ----------- |
| `cargo build` | Build the project |
| `cargo run` | Run the server |
| `cargo test` | Run tests |
| `cargo fmt --check && cargo clippy` | Lint & format check |

### Frontend

| Command | Description |
| ------- | ----------- |

| `npm run check` | Type-check the project |
| `npm run lint` | Lint the codebase |
| `npm run format` | Format code with Prettier |
| `npm run test:unit` | Run unit tests |

## API

This project uses OpenAPI v3 for API documentation. The spec is generated by the backend (`openapiv3` crate). During development, the frontend proxies `/api` requests to the backend server.
